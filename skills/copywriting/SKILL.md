## Core Principles

### The Copywriting Hierarchy
1. **Attention** - Stop the scroll, earn the read
2. **Interest** - Connect to their problem/desire
3. **Desire** - Show the transformation
4. **Action** - Make the next step clear

### Benefit > Feature
```
Feature: 256-bit encryption
Benefit: Your data stays private

Feature: 24/7 support
Benefit: Get help whenever you need it

Feature: AI-powered
Benefit: Save 10 hours every week
```

### Reader-Centric Language
```
❌ We're excited to announce...
✓ You can now...

❌ Our product features...
✓ Get [benefit] with...

❌ We believe in quality...
✓ Built to last you years...
```

## Headline Formulas

### Problem-Solution
```
Tired of [problem]? [Solution] in [timeframe].
Stop [pain point]. Start [desired outcome].
[Problem] keeping you up at night? Here's the fix.
```

### Benefit-Focused
```
[Achieve outcome] without [common obstacle]
The fastest way to [desired result]
[Number]% of [audience] [achieve result]. Here's how.
```

### Curiosity/Intrigue
```
The [unexpected adjective] way to [achieve goal]
Why [common approach] is costing you [loss]
What [successful people] know about [topic]
```

### Social Proof
```
How [company/person] [achieved result]
[Number] [people] can't be wrong
Join [number] [audience] who [outcome]
```

### Direct/Command
```
Get [result] now
Start [action] today
Claim your [offer]
```

## Body Copy Structure

### PAS (Problem-Agitate-Solve)
```markdown
**Problem**: You know the feeling...
[Describe the problem they're experiencing]

**Agitate**: And it gets worse...
[Amplify the pain, show consequences]

**Solve**: There's a better way...
[Introduce your solution]
```

### AIDA (Attention-Interest-Desire-Action)
```markdown
**Attention**: [Headline that stops them]

**Interest**: [Connect to their situation]
You've probably tried [alternatives]...

**Desire**: [Show the transformation]
Imagine [ideal outcome]...

**Action**: [Clear CTA]
Start your free trial today.
```

### Before-After-Bridge
```markdown
**Before**: [Current painful state]
Right now, you're probably...

**After**: [Desired future state]
But imagine if you could...

**Bridge**: [Your solution]
[Product] makes this possible by...
```

## CTA Best Practices

### Action-Oriented
```
✓ Start my free trial
✓ Get the guide
✓ Show me how
✓ Claim my spot

❌ Submit
❌ Click here
❌ Learn more (vague)
```

### Value-Focused
```
✓ Get instant access
✓ Save 20% today
✓ Join 10,000+ members

❌ Buy now
❌ Sign up
❌ Register
```

### First Person
```
✓ Start my free trial (user perspective)
✓ Yes, I want this
✓ Send me the checklist

❌ Start your free trial (less engaging)
```

### Reduce Friction
```
✓ Start free trial — no credit card required
✓ Get the guide — instant download
✓ Book a call — just 15 minutes
```

## Copy Length Guidelines

### Headlines
- Primary: 6-12 words ideal
- Subheadlines: 12-20 words
- Can be longer if curiosity-driven

### Body Paragraphs
- 2-4 sentences max
- One idea per paragraph
- Break up with subheads every 2-3 paragraphs

### CTAs
- 2-5 words for buttons
- Can add supporting text below

### General Rule
- Long copy for high-consideration purchases
- Short copy for low-friction actions
- Always scannable regardless of length

## Tone Adjustments

### By Audience
| Audience | Tone Shift |
|----------|------------|
| B2B Enterprise | Professional, data-driven |
| B2B Startup | Conversational, ambitious |
| B2C Premium | Aspirational, refined |
| B2C Mass Market | Friendly, relatable |
| Technical | Precise, credible |

### By Funnel Stage
| Stage | Approach |
|-------|----------|
| Awareness | Educational, problem-focused |
| Consideration | Benefit-focused, comparative |
| Decision | Proof-heavy, urgency |
| Retention | Value-reinforcing, helpful |

## Power Words

### Urgency
Now, Today, Instant, Fast, Quick, Limited, Deadline, Don't miss

### Value
Free, Save, Bonus, Exclusive, Premium, Best, Proven, Guaranteed

### Emotion
Love, Hate, Fear, Joy, Discover, Secret, Revealed, Breakthrough

### Trust
Proven, Trusted, Certified, Guaranteed, Research-backed, [Number] customers

## Common Mistakes

### Avoid
- Starting with "Welcome to..."
- Feature lists without benefits
- Jargon the audience doesn't use
- Weak CTAs (Learn more, Submit)
- Wall of text with no structure
- Writing about yourself, not them

### Fix Patterns
| Instead of | Write |
|------------|-------|
| We offer... | You get... |
| Our platform... | With [product], you can... |
| Features include... | This means you can... |
| Click here to... | [Specific action verb]... |
