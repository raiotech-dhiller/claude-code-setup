## Testing Headlines

### A/B Test Variables
1. Formula type (benefit vs curiosity vs social proof)
2. Specificity (numbers vs vague)
3. Audience callout (included vs omitted)
4. Length (short vs long)
5. Power words (with vs without)

### Quick Headline Checklist
- [ ] Clear benefit or curiosity hook
- [ ] Specific (numbers, timeframes if relevant)
- [ ] Audience-appropriate language
- [ ] No jargon or confusion
- [ ] Matches content that follows
- [ ] Would I click on this?

## Headlines by Context

|---------|---------------|

| Landing page | Benefit, Outcome |

| Social ad | Question, Problem, Hook |
| Search ad | Keyword + Benefit |
| Sales page | Bold claim + Proof |
