# Landing Page Copywriting

## Landing Page Anatomy

### Above the Fold
```
┌─────────────────────────────────────────┐
│  Logo                      Nav (minimal) │
├─────────────────────────────────────────┤
│                                         │
│     HEADLINE                            │
│     Subheadline supporting the promise  │
│                                         │
│     [Primary CTA]  [Secondary CTA]      │
│                                         │
│     Social proof snippet                │
│                                         │
└─────────────────────────────────────────┘
```

### Full Page Structure
1. **Hero** - Headline, subhead, CTA, social proof
2. **Problem** - Articulate their pain
3. **Solution** - Introduce your approach
4. **Benefits** - 3-5 key benefits with proof
5. **How It Works** - Simple 3-step process
6. **Social Proof** - Testimonials, logos, stats
7. **Objection Handling** - FAQ or address concerns
8. **Final CTA** - Repeat offer with urgency

## Hero Section Copy

### Headline Approaches

**Outcome-focused:**
```
Get [specific outcome] in [timeframe]
[Outcome] without [common sacrifice]
The [adjective] way to [achieve result]
```

**Problem-focused:**
```
Stop [painful activity] forever
Never [frustration] again
Finally, a solution to [problem]
```

**Audience-focused:**
```
For [audience] who want [outcome]
Built for [audience] who [characteristic]
The [product type] [audience] actually use
```

### Subheadline Patterns
```
[Product] helps [audience] [achieve outcome] so they can [bigger benefit].

Join [number] [audience] who [outcome] with [product].

The only [product type] that [key differentiator].
```

### Hero CTA + Supporting Text
```
[Start Free Trial]
No credit card required • Setup in 2 minutes

[Get the Guide]
Join 50,000+ subscribers

[Book a Demo]
30-minute call • No pressure
```

## Problem Section

### Structure
```markdown
## You know the feeling...

[Describe situation they're in - use "you"]

Every day, you [painful activity].
You've tried [alternative 1], but [why it failed].
You've tried [alternative 2], but [why it failed].

Meanwhile, [consequence of unsolved problem].

**Sound familiar?**
```

### Example
```markdown
## Drowning in spreadsheets?

You started with one spreadsheet. Then another. Now you're managing 
a maze of files, praying nothing breaks.

You've tried project management tools, but they're overkill for your team.
You've tried shared docs, but version control is a nightmare.

Meanwhile, your actual work keeps piling up.

**There's a better way.**
```

## Solution Section

### Structure
```markdown
## Introducing [Product]

[One sentence: what it is and who it's for]

[Product] [core value proposition].

**Finally, you can:**
- [Benefit 1]
- [Benefit 2]
- [Benefit 3]
```

### Example
```markdown
## Introducing TaskFlow

The work management tool built for teams who hate work management tools.

TaskFlow gives you just enough structure to stay organized—without the 
bloat that slows you down.

**Finally, you can:**
- See everyone's work in one place
- Ship projects on time (without the stress)
- Actually enjoy your Monday mornings
```

## Benefits Section

### Benefit Block Template
```markdown
### [Benefit-focused headline]

[2-3 sentences explaining the benefit and how it works]

**The result:** [Specific outcome with proof point if possible]
```

### Example
```markdown
### Save 10+ hours every week

Our smart automation handles the busywork—status updates, reminders, 
reports—so you can focus on work that matters.

**The result:** Teams report saving an average of 12 hours per week 
on administrative tasks.
```

### Benefit Writing Tips
- Lead with outcome, not feature
- Be specific (numbers, timeframes)
- Include proof when possible
- Keep each benefit to 3-4 sentences

## How It Works Section

### 3-Step Formula
```markdown
## How it works

**1. [Action verb] your [noun]**
[One sentence explaining this step]

**2. [Action verb] your [noun]**
[One sentence explaining this step]

**3. [Action verb] your [noun]**
[One sentence explaining this step]

[CTA: Get started in minutes]
```

### Example
```markdown
## How it works

**1. Connect your tools**
Link your existing apps in one click—no developer needed.

**2. Set your workflow**
Choose from templates or build your own in minutes.

**3. Watch it run**
TaskFlow handles the rest while you focus on real work.

[Start your free trial]
```

## Social Proof Section

### Testimonial Format
```markdown
"[Specific result or transformation they experienced]"

**[Name], [Title] at [Company]**
[Optional: relevant context like company size or industry]
```

### Effective Testimonials Include
- Specific results (numbers, timeframes)
- Before/after context
- Relatable role/company
- Genuine voice (not too polished)

### Logo Bar Copy
```markdown
**Trusted by 2,000+ teams**

**Join teams from**
[Logos]
```

## FAQ/Objection Section

### Common Objections to Address
- Is it worth the price?
- Is it hard to set up?
- Will it work for my use case?
- What if I need help?
- Can I cancel/get a refund?

### FAQ Format
```markdown
## Questions? We've got answers.

**Is [Product] right for my team size?**
[Answer that addresses the concern and pivots to benefit]

**How long does setup take?**
[Answer with specific timeframe + support available]

**What if I need help?**
[Answer highlighting support options]
```

## Final CTA Section

### Structure
```markdown
## [Benefit-focused headline]

[One sentence reinforcing the value]

[Primary CTA]

[Risk reducer: guarantee, free trial, etc.]
```

### Example
```markdown
## Ready to get your time back?

Join 10,000+ teams who've simplified their workflow with TaskFlow.

[Start Free Trial — No Credit Card Required]

30-day money-back guarantee. Cancel anytime.
```

## Page-Specific Variations

### SaaS Landing Page
- Emphasize: Time savings, efficiency, integration
- Proof: Logos, user counts, performance stats
- CTAs: Free trial, demo, pricing

### Service Landing Page
- Emphasize: Expertise, results, process
- Proof: Case studies, testimonials, credentials
- CTAs: Book call, get quote, contact

### Lead Magnet Landing Page
- Emphasize: Value of content, quick win
- Proof: Download counts, testimonials
- CTAs: Get the guide, download now

### E-commerce Product Page
- Emphasize: Benefits, quality, satisfaction
- Proof: Reviews, ratings, sold count
- CTAs: Add to cart, buy now
