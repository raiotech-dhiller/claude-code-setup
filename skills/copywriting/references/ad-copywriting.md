# Ad Copywriting

## Platform Specifications

### Meta (Facebook/Instagram)
| Element | Character Limit | Best Practice |
|---------|-----------------|---------------|
| Primary text | 125 visible (more with "see more") | Hook in first line |
| Headline | 27 chars visible | Benefit-focused |
| Description | 27 chars visible | Supporting info |
| CTA button | Platform options | Match to goal |

### Google Ads (Search)
| Element | Character Limit |
|---------|-----------------|
| Headline 1-3 | 30 chars each |
| Description 1-2 | 90 chars each |
| Display URL path | 15 chars each |

### Google Ads (Display/YouTube)
| Element | Character Limit |
|---------|-----------------|
| Short headline | 25 chars |
| Long headline | 90 chars |
| Description | 90 chars |

### LinkedIn
| Element | Character Limit |
|---------|-----------------|
| Intro text | 150 chars visible | 
| Headline | 70 chars |
| Description | 100 chars |

### Twitter/X
| Element | Character Limit |
|---------|-----------------|
| Tweet text | 280 chars |
| Card headline | 70 chars |
| Card description | 200 chars |

## YouTube Ads

### Hook Formula (First 5 seconds)
```
[Pattern interrupt] + [Problem/curiosity] + [Promise]

"Stop! If you're [audience doing thing], you need to see this."
"What if I told you [surprising claim]?"
"[Audience], here's the mistake you're making..."
```

### Script Structure (15-30 seconds)
```
0-5s: Hook (must earn the skip)
5-15s: Problem + agitate
15-25s: Solution + benefits
25-30s: CTA + offer
```

### Text Overlay Guidelines
- Large, readable text
- One message per frame
- Reinforce audio (don't duplicate)
- Include CTA text

## Ad Copy Best Practices

### By Funnel Stage

**Top of Funnel (Awareness):**
- Lead with problem or curiosity
- Educational content
- Softer CTAs (Learn more, See how)
- Focus on relevance over selling

**Middle of Funnel (Consideration):**
- Lead with solution/benefit
- Social proof and comparisons
- Stronger CTAs (Get started, Try free)
- Address objections

**Bottom of Funnel (Decision):**
- Lead with offer/urgency
- Heavy social proof
- Direct CTAs (Buy now, Sign up)
- Scarcity and risk reducers

### Audience-Specific Adjustments

| Audience | Adjust |
|----------|--------|
| Cold | More education, softer ask |
| Warm | More benefits, medium ask |
| Hot | More urgency, direct ask |
| Retargeting | Address abandonment, special offer |

## Testing Framework

### Elements to Test
1. Hook/opening line
2. Angle (problem vs benefit vs story)
3. CTA copy
4. Headline variations
5. Image/video (paired with copy)

### Ad Variations Template
```
Version A: Problem angle
Version B: Benefit angle
Version C: Social proof angle
Version D: Curiosity angle
```

### Iteration Process
1. Test angles first (biggest impact)
2. Winner: test hooks
3. Winner: test CTAs
4. Winner: test details
5. Scale winner, repeat

## Common Mistakes

### Avoid
- Too much text (especially on image)
- Weak or generic hooks
- Features without benefits
- No clear CTA
- Mismatch between ad and landing page
- Ignoring platform character limits

### Quick Fixes
| Problem | Fix |
|---------|-----|
| Low CTR | Stronger hook, clearer benefit |
| High CTR, low conversion | Align ad with landing page |
| Low relevance score | Better audience targeting + copy match |
| Ad fatigue | New creative angle |
